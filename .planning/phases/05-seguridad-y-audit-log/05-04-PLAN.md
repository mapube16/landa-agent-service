  <action>
    In `app/features/qa/nodes.py`:

    1. Add module import `from app.security import audit_log` (module-style so a single monkeypatch of `audit_log.emit_task` covers all call sites).
    2. In `node_answer`, derive once: `conv_id = state.get("thread_id") or state.get("conversation_id")` and `poliza = state.get("poliza_id")` (read state.py to pick the fields that actually exist — use whichever identifier node_answer already has access to).
    3. After the conversation `llm.ainvoke` returns: `audit_log.emit_task(action="llm_turn", actor="bot", conversation_id=conv_id, poliza_id=poliza, payload={"model_role": "conversation", "response_sha256": hashlib.sha256(str(response.content).encode()).hexdigest(), "has_tool_calls": bool(getattr(response, "tool_calls", None))})`.
    4. Inside the tool execution block (after `tool_node.ainvoke`): `audit_log.emit_task(action="tool_call", actor="bot", conversation_id=conv_id, poliza_id=poliza, payload={"tools": ",".join(tc.get("name", "") for tc in response.tool_calls)})`.
    5. After `judge_response` returns the rubric: `audit_log.emit_task(action="judge_decision", actor="judge", conversation_id=conv_id, poliza_id=poliza, payload={"approved": is_approved(rubric), **{f"flag_{k}": bool(v) for k, v in rubric.model_dump().items()}})` — flatten rubric booleans; coerce every value to bool/str (AuditPayload constraint).
    6. In `node_escalate` (entry): `audit_log.emit_task(action="escalation", actor="bot", conversation_id=..., poliza_id=..., payload={"reason": str(reason)})` using the same reason var it already logs.

    Rules: emit_task is sync and never raises — call it inline, do NOT await, do NOT wrap in extra try/except (05-01 guarantees). Never put message text in payloads — hashes only (no-PII invariant; payload is hashed anyway but keep discipline). Do not reorder existing logic; hooks are additive lines.

    Create `tests/security/test_audit_capture.py` implementing the behavior bullets. Reuse the 05-02 fake-LLM/fake-judge harness pattern (if 05-02 already merged, import nothing from it — copy the small harness; test files stay independent).

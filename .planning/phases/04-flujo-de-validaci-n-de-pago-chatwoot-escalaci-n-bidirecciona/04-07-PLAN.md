<objective>
Implement the lambda-proyect handoff endpoint `POST /case/handoff/no_answer` (D-19, D-20, D-21, D-22, D-23). This is the only F4 inbound from a non-Meta non-Chatwoot source, and it's the trigger for the Meta UTILITY template that re-opens the WhatsApp channel when the voice agent failed to contact the deudor.

Purpose: Without this endpoint, lambda-proyect's voice agent has no way to escalate to WhatsApp follow-up.

Output:
- New router `app/webhooks/handoff.py` with bearer-token auth, Pydantic body validation, case UPSERT, and template send.
- Router registered in `app/main.py`.
- Tests for auth reject, idempotent UPSERT, template payload shape.
</objective>

    Define request model `NoAnswerHandoff(BaseModel)`:

    Dependency: `def _verify_bearer(authorization: str | None = Header(None))`:
    - If `authorization` is None or does not start with `"Bearer "`: raise HTTPException(401, "missing bearer").
    - Strip the prefix; `provided = authorization[7:].encode()`.
    - `expected = Settings().lambda_proyect.internal_token.get_secret_value().encode()`.
    - If `not hmac.compare_digest(provided, expected)`: raise HTTPException(401, "invalid bearer").
    - Return None (the dependency only gates).

    Endpoint:
    `@router.post("/handoff/no_answer", dependencies=[Depends(_verify_bearer)])`
    `async def handoff_no_answer(body: NoAnswerHandoff, request: Request) -> dict`:

    - Open async session via `async with db() as session:`.
    - `existing = (await session.execute(select(Case).where(Case.case_id == body.case_id))).scalar_one_or_none()`.
    - If `existing` is None:
      - Insert Case: `case_id=body.case_id, phone=body.phone, cliente_nombre=body.cliente_nombre, poliza_id=body.numero_poliza, status="awaiting_receipt"`. Commit.

      - Log info `handoff.no_answer.template_sent` with `case_id=str(body.case_id)`, `phone_hash` (use the existing `_hash_phone` from `meta_cloud.py`).
      - Return `{"case_id": str(body.case_id), "sent": True}`.
    - Else: log info `handoff.no_answer.idempotent_skip`, return `{"case_id": str(body.case_id), "sent": False}`.

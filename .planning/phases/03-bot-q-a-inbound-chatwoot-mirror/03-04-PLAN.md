    12. `r"reveal\s+(your|the)\s+(system|hidden)\s+(prompt|instructions)"` (re.I)

    Función `def sanitize(text: str) -> SanitizeResult`:
    - Step 1 NFKC: `normalized = unicodedata.normalize("NFKC", text)`.
    - Step 2 strip invisible: `cleaned = INVISIBLE_CHARS_PATTERN.sub("", normalized)`.
    - Step 3 strip control: `cleaned = CONTROL_CHARS.sub("", cleaned)`.
    - Step 4 length cap: `if len(cleaned) > MAX_LENGTH: return SanitizeResult(blocked=True, reason="length_exceeded")`.
    - Step 5 pattern match: for pattern in INJECTION_PATTERNS: if pattern.search(cleaned): return blocked with reason=`f"pattern_match:{pattern.pattern[:30]}"` (truncate pattern para no leakear full regex en logs si reason termina logged).
    - Log: `log.info("prompt_firewall.scan", blocked=result.blocked, reason=result.reason)` SOLO si blocked=True (no flood con benign calls).
    - Return `SanitizeResult(blocked=False, cleaned=cleaned)`.
